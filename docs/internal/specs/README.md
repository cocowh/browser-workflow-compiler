# Capability Specs

Specs combine requirements, solution design, risks, and acceptance criteria for a specific capability.

## Naming

Use date plus capability name:

```text
YYYY-MM-DD-capability-name.md
```

Example:

```text
2026-06-12-local-recording.md
```
